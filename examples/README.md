# examples

The `examples` directory contains small runnable applications that show the intended style of `go-bootstrap`.

- `simpleapi`
  - Minimal server bootstrap with `Module`, typed `StartStop`, and `In`
- `overrideapp`
  - Base composition plus `Override` for environment-specific entrypoints
- `simplecli`
  - Minimal CLI bootstrap with `HookFunc`
- `modularapp`
  - Larger module composition with multiple entry param structs
